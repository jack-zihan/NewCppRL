    """
    完整的死循环分析：
    
    数学模型：
    设农田面积为A，已放置杂草数为n，总需求为N
    单次尝试成功概率：P(n) = (A - n) / (W × H)
    期望尝试次数：E[attempts] = 1 / P(n)
    
    当n接近A时：
    - P(n) → 0
    - E[attempts] → ∞
    - 死循环！
    
    具体计算：
    - 地图400×400，农田30% = 48,000格
    - 要求50,000个杂草
    - P(48000) = 0/160000 = 0
    - 结果：必然死循环
    """

    weed_count = 0  # 当前已放置的杂草数量
    
    # L736: 计算实际需要的杂草数

        # L737: 浮点数表示比例
