方向场单位制和方向映射分析报告

  1. 单位制：弧度制 (Radians)

  通过代码分析确认：
  - 使用弧度制：angles = np.linspace(0, pi, num_filters, endpoint=False)
  - 范围：[0, π) 弧度，对应 [0°, 180°)
  - 原因：方向场表示的是无向的方向，0°和180°是等价的（都表示水平方向）

  2. 关键转换公式

  # HSV色相映射（0-1范围）
  hsv_hue = orientations / np.pi

  # 角度转换（弧度→度）
  degrees = orientations * (180 / np.pi)

  # 灰度映射（0-255范围）
  gray = orientations * (255 / np.pi)

  3. 方向与钟点对应关系（图像坐标系）

  在图像处理中，Y轴向下，因此方向映射如下：

  | 弧度值  | 角度   | 钟点方向 | 实际方向     | HSV色相   |
  |------|------|------|----------|---------|
  | 0    | 0°   | 3:00 | 水平向右 →   | 0.0 (红) |
  | π/6  | 30°  | 4:00 | 右下斜 ↘    | 0.167   |
  | π/4  | 45°  | 4:30 | 45°斜向 ↘  | 0.25    |
  | π/3  | 60°  | 5:00 | 60°斜向 ↘  | 0.333   |
  | π/2  | 90°  | 6:00 | 垂直向下 ↓   | 0.5 (青) |
  | 2π/3 | 120° | 7:00 | 左下斜 ↙    | 0.667   |
  | 3π/4 | 135° | 7:30 | 135°斜向 ↙ | 0.75    |
  | 5π/6 | 150° | 8:00 | 150°斜向 ↙ | 0.833   |
  | π    | 180° | 9:00 | 水平向左 ←   | 1.0 (红) |

  4. Gabor滤波器的角度调整

  代码中有一个重要细节：
  gabor_kernel = gabor_kernel_2d(sigma_u, sigma_v, angle + pi/2, lambd)
  这里加了 π/2（90度），可能是为了：
  - 将方向从水平基准调整为垂直基准
  - 或者适应特定的纹理方向定义

  5. 重要特性

  1. 无向性：方向场范围 [0, π) 表示无向的方向
    - 0° ≡ 180°（水平方向）
    - 90°（垂直方向）是唯一的
  2. HSV颜色编码：
    - 色相(Hue) = orientation / π
    - 饱和度(Saturation) = 1.0（固定）
    - 明度(Value) = 1.0（固定最大亮度）
  3. 可视化含义：
    - 红色（Hue=0或1）：水平方向
    - 青色（Hue=0.5）：垂直方向
    - 渐变色：中间角度

  6. 实际应用建议

  - 水平纹理：orientation ≈ 0 或 π
  - 垂直纹理：orientation ≈ π/2
  - 45°斜纹理：orientation ≈ π/4
  - 135°斜纹理：orientation ≈ 3π/4

  这个方向场系统非常适合分析木材、织物等具有明显方向性的纹理材料。
